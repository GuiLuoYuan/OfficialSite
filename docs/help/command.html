◎ 符号含义参照


  [***] - 非必填参数
  <***> - 必填参数
  ***/*** - 多种参数可选


◎ 传送相关命令


  /back - 返回上一个传送点或死亡点。
  /tpa <玩家> - 请求传送至指定玩家所在位置
  /tpahere <玩家> - 请求指定玩家传送到你所在位置
  /tpaccept - 接受其他玩家向你发送的传送申请
  /tpdeny - 拒绝其他玩家向你发送的传送申请
  /tpacancel - 取消所有你向其他玩家发送的传送请求
  /home <名称> - 传送到名字为 家 的 Home 传送点
  /sethome <名称> - 添加名字为 家 的 Home 传送点
  /delhome <名称> - 删除名字为 家 的 Home 传送点
  /warps - 查看所有 warp 传送点
  /warps <名称> - 传送到指定的 Warp 传送点


◎ 快捷操作相关指令


  /hat - 将手持物品作为帽子带上
  /trash - 打开垃圾桶菜单
  /enderchest - 打开便携末影箱
  /workbench - 打开便携工作台
  注意事项：打开便携末影箱的指令 /enderchest 可缩写为 /ec ；打开便携工作台的指令 /workbench 可缩写为 /wb 。


◎ 肢体动作相关命令


  /sit - 在当前位置坐下
  /lay - 在当前位置躺下
  /bellyflop - 在当前位置趴下
  /crawl - 启用/关闭爬行模式


◎ 聊天系统相关指令


  /message <玩家> <信息> - 向指定玩家发送私聊信息
  注意事项：向指定玩家发送私聊信息的指令 /message 也可使用这三个变种 /tell 、/w 与 /msg 。
  其他事项：本条目还没更新完毕，敬请谅解！


◎ 签到系统相关命令


  /signin gui - 打开 归落原 Minecraft Group Server 签到菜单
  /signin click [日期] - 为今日签到或对指定日期进行补签
  /signin leaderboard [日期] [页码] - 查询今日或指定日期的签到排行榜

◎ 创作积分相关命令


  /balance - 查看自己的创作积分点数
  /balancetop - 查看 归落原 Minecraft Group Server 创作积分点数排行榜
  /pay <玩家> <数额> - 向目标玩家转移指定数额的创作积分点数


◎ 查询信息相关命令


  /season - 查看当前季节详情
  /slime - 查阅玩家自身半径 5 格以内的史莱姆区块


◎ 特殊需求相关命令


  /asb toggle [on/off] - 切换信息显示栏为显示或隐藏状态
  /suicide - 自杀丨再见，残酷的世界
  注意事项：现实世界请勿模仿！
